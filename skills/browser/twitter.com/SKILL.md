# Twitter / X — ver skill x.com

Twitter redirige a x.com. Seguí exactamente la misma guía de la skill `x.com`.

## Redirección

Si navegás a `twitter.com`, serás redirigido a `x.com`. Todas las URLs y flujos son idénticos — usá `x.com/...` directamente para evitar redirecciones innecesarias.

## Login

1. Navegá directamente a `https://x.com/i/flow/login`
2. Seguí el flujo de login documentado en la skill x.com.

## Extracción, búsqueda y demás

Idéntico a la skill x.com. Las URLs son intercambiables (`twitter.com/handle` == `x.com/handle`).

## Límites

Idénticos a x.com — no publiques, no interactúes, solo extraé.
